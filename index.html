¿COMO HACER PAGINAS CON HTML?

EN ESTA PAGINA, APRENDEREMOS COMO REALIZAR PAGINAS WEB CON HTML - APRENDIZ STEFANI ALVAREZ


HTML significa Lenguaje de Marcas de Hipertexto. No es un lenguaje de programación per se, sino un lenguaje de marcado que se utiliza para crear la estructura de las páginas web. Define el significado y la estructura del contenido web.

  1. ESTRUCTURA BÁSICA DE UN DOCUMENTO HTML

    2. Todo documento HTML comienza con una declaración de tipo de documento, seguido por elementos que definen la cabeza (head) y el cuerpo (body) de la página.

  2. TITULOS Y VIÑETAS

    3. Cuando pones un texto en una etiqueta (h1), el texto estará en negrilla, y la dimensión de la letra será igual al número del encabezado. Los títulos pueden tener dimensiones entre 1 y 6. Las viñetas en HTML ofrece dos tipos principales de listas:

    • Listas desordenadas (con viñetas)
    • se utiliza con UL
    1. Listas ordenadas (numeradas)
    2. se utiliza con OL

  3. ATRIBUTOS

    4. Los elementos HTML pueden tener atributos que proporcionan información adicional sobre el elemento. Por ejemplo, el elemento (img) tiene atributos como src (que indica la fuente de la imagen) y alt (que proporciona texto alternativo):

  4. BORDES

    5. Para poder indicar un borde determinado con un color y un grosor concreto para un div, dentro del código CSS, utilizamos la propiedad border, que utiliza el siguiente formato:
    border: 2px solid grey;

  5. ELEMENTOS Y ETIQUETAS

    6. HTML utiliza "etiquetas" para marcar el inicio y el fin de los elementos. Estas etiquetas generalmente vienen en pares, por ejemplo:

    ETIQUETA DESCRIPCIÓN
    head Texto no visible (metadatos, codigos, palabras claves)
    title Define el título de la página del navegador.
    body Encierra todo el contenido visible de la página
    ul, ol, li UL (viñetas), OL (números), y LI para los ítems.
    div y span Contenedor para agrupar contenido de bloque y de linea, utilizado CSS.

  6. FORMULARIOS

    7. Los formularios en HTML son elementos esenciales para recopilar información de los usuarios en una página web. Permiten enviar datos a un servidor para su procesamiento.






  7. ENLACES

    8. Los enlaces (o hipervínculos) en HTML son elementos fundamentales que permiten navegar entre páginas web y recursos. Se crean con la etiqueta (a), con href se coloca el link, y con target="_blank" se abre el link en otra página.
